# Rooftop Solar Analysis Tool

## Setup Instructions

1. Install Python (>=3.10) from https://www.python.org/downloads/
2. Open terminal or command prompt.
3. Navigate to the project folder.
4. Run: pip install -r requirements.txt
5. Run: streamlit run main_app.py

## Description

This tool lets users upload a satellite rooftop image and receive:
- Estimated usable area for solar
- Suggested solar panel type
- ROI and payback details

## Example Use Case

A homeowner uploads an image of their rooftop and receives recommendations on how much solar they can install and expected financial returns.
